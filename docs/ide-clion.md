# CLion/JetBrains Tips

- Use the bundled CMake and set generator to Ninja.
- Enable `clang-tidy` inspections and point to project `.clang-tidy`.
- Set `-DCMAKE_EXPORT_COMPILE_COMMANDS=ON` for full indexing.
