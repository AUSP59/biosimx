# Versioning Strategy
- Semantic Versioning (SemVer) is used for public APIs and the CLI.
- Patch: bug/security; Minor: backward-compatible features; Major: breaking changes.
- Pre-releases use `-rc.N` suffix; release candidates undergo full CI with stricter gates.
