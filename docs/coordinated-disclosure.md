# Coordinated Vulnerability Disclosure

- Report privately to the security contact listed in `SECURITY.md`.
- We acknowledge within 72h and provide a remediation timeline.
- Fixes are validated under embargo; we coordinate a CVE if applicable.
- On release, we publish advisories with CVSS vector and mitigation steps.
